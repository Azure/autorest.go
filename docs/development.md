# Prerequisites

- [npm](https://www.npmjs.com/)
- npm install -g "@microsoft/rush"
- npm install -g typescript
- npm install -g autorest

# Building

## To build autorest.go

```terminal
rush build
```

## To regenerate test and samples

```terminal
rush regenerate
rush regenerate --filter TestName
```
